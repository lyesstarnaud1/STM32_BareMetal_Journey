# STM32_BareMetal_Journey
Mastering stm32 (F446re) through bare-metal "C" programming.Applying my mathemathical logic and my experience in hardware repairs to build high performance projects
 
### 🚀 About This Journey
I am a **Mathematics Graduate** and **Hardware Repair Specialist** transitioning into **Embedded Systems Engineering**. This repository documents my progress in mastering the STM32 ARM Cortex-M4 architecture (F446RE) using bare-metal C programming without HAL or external libraries.

### 🛠 Tools of the Trade
- **Microcontroller:** STM32F446RE (Nucleo-64)
- **OS:** Debian 12 (Linux)
- **Debugger:** FNIRSI 2C53T (2-Channel Oscilloscope)
- **Compiler:** ARM GNU Toolchain / STM32CubeIDE
